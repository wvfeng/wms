## 介绍

http://www.thinkphp.cn/code/2220.html

角色管理、权限管理、管理员、门店管理、属性管理、职位管理、人员管理、购货、销售、库存管理 、个人信息... 


本系统采用TP框架，并对一些的页面实行了静态化。一些不会改变的数据采用了memcache保存，比如以往的销售记录 购货记录 ...


为了提高web交互的友好度，所有操作 是基于ajax实现（包括 表单验证 添加 搜索 提示...）
